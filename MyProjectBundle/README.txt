< 제작 방법 >
이클립스의 다이나믹 웹 프로젝트로 제작.
데이터베이스 테이블은 MySQL 워크밴치로 제작.

--------------------------------------------------------

< 프로젝트 구성 >

src 폴더 하위 :
[webapp 폴더]
- 메인페이지, 상단 menu 바, 하단 footer 등 기본적인 jsp 페이지들
- project 폴더 
	(프로젝트 등록, 프로젝트 목록, 프로젝트 상세 페이지, 프로젝트 등록 처리 등의 프로젝트와 관련된 jsp 파일들이 들어있다)
- member 폴더 
	(회원가입, 로그인, 로그아웃, 로그인 처리 등의 회원 정보와 관련된 jsp 파일들이 들어있다)
- resources 폴더 
	(css 폴더, js 폴더, 이미지 폴더가 들어있다)
[java 폴더]
- bundle 폴더
	(다국어 지원을 위한 properties 페이지들이 들어있다)

sql 폴더 하위 :
project, projectImage, member 테이블 정보를 기록해두었다

(데이터베이스는 Export 하여 프로젝트 폴더 외부의 DB 폴더에 따로 빼두었다
 워크밴치에 Import 하여 사용할 수 있다)

------------------------------------------------------------

< 프로젝트 실행 >
webapp 폴더의 mainpage.jsp 파일을 실행시키면 동작한다
